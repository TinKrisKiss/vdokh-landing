# Лендинг для бьюти-пространства «ВДОХ»

**Дизайн + Вайб-кодинг | 2025**

🎯 Задача
Создать атмосферный, быстрый сайт для салона красоты, который передаёт вайб лёгкости, собирает заявки без бэкенда и адаптирован под мобильные.

🛠 Что сделано
• Разработан фирменный стиль и брендбук (палитра, типографика, Tone of Voice)
• Сгенерирован визуальный контент через AI (Midjourney) с единой стилистикой
• Сверстан адаптивный многостраничный лендинг (HTML + Tailwind CSS + Alpine.js)
• Подключена форма обратной связи через Formspree (без сервера, на почту)
• Реализован лайтбокс-слайдер для галереи работ
